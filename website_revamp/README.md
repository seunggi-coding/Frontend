# 한국사이버복지대학교 웹사이트 리뉴얼 프로젝트 🖍️

## 소개

**(현)한국사이버복지대학교 (구)한국사이버복지대학**의 웹사이트 리뉴얼 프로젝트입니다.  
기존 사이트를 현대적이고 사용자 친화적인 인터페이스로 개선하고, 반응형 디자인을 적용하여 접근성을 높이는 것을 목표로 진행되었습니다.

본 프로젝트는 아래와 같은 내용을 포함하고 있습니다:
- 프로젝트 개요
- 화면 구성 및 스크린샷
- 사용 기술 스택
- 프로젝트 팀원
